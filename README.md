# 🚀 AutoIME-Matrix (跨平台智能输入法强控引擎)

> 天下苦 Windows / macOS 输入法频繁切换久矣！
> 这是一个旨在彻底终结“中英文状态错乱”、“Shift键薛定谔记忆”的底层强控切换神器。进软件自动切英文，退出来自动恢复中文。

## ✨ 核心亮点与痛点解决

* 🚫 **无脑防错乱**：进入研发与设计软件（如 Cursor, VSCode, IDEA, Figma, Terminal 等）自动强锁英文档；回到通讯软件（微信、钉钉）瞬间切回中文。
* ⚡ **极低系统消耗**：摒弃低效的死循环扫描，采用底层系统窗口焦点事件驱动，无论 Win 还是 Mac，CPU 占用率几乎永远为 **0.00%**。
* 🛡️ **免疫竞态条件 (Windows)**：独创“延迟双杀 (Double Tap)”强行覆盖算法，彻底抹杀 Windows 极其恶心的原生 Shift 记忆 Bug。
* 🌐 **浏览器沙盒免检 (Windows)**：精准豁免 Chrome / Edge 等 Chromium 内核应用，将其输入法控制权交还给系统，彻底解决浏览器内输入卡顿的坑。

## 📦 下载与安装指南 (小白必看！)

> ⚠️ **避坑警告：**
> 请**不要**直接点击页面上方的绿色 `Code` 按钮下载 ZIP 源码！源码内不含运行环境，直接运行必定报错。源码仅供极客阅读和二次开发。
> **普通用户请务必前往右侧的 [Releases (发行版)](#) 页面下载打包好的完整安装包！**

### 🍎 macOS 用户 (一键离线尊享版)
为了解决国内访问外网环境极其缓慢的痛点，本工具已将底层依赖引擎（Hammerspoon）一并打包在压缩包内（约 41MB），全程离线安装，开箱即用。

1. 前往页面右侧的 **[Releases (发行版)](#)** 区域。
2. 下载最新版本的 `Mac-EN-Input-AutoSwitcher_vX.X.zip`。
3. 下载后双击解压压缩包。
4. 打开 Mac 自带的 **终端 (Terminal)**，运行解压出来的 `install.sh` 脚本（也可以直接将脚本拖入终端回车）。
5. 按照提示赋予系统辅助功能权限，即可体验带有右上角状态栏联动的极速智能切换！

### 🪟 Windows 用户 (原生典藏版)
1. 前往页面右侧的 **[Releases (发行版)](#)** 区域。
2. 下载最新版本的 `Win-EN-Input-AutoSwitcher_vX.X.bat` 脚本文件。
3. **右键点击该文件 -> 选择“以管理员身份运行”**。
4. 在弹出的交互黑窗口中输入 `2`。
5. 脚本会自动在后台为你即时编译出纯原生的 EXE 守护进程，并静默配置好开机自启。看到成功提示后关闭窗口，尽享丝滑！
